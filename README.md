# HyperSR
Pytorch implementation of " Deep hyperspectral image super-resolution with transformers (基于 Transformer的高光谱图像超分辨率重建)", SCIENTIA SINICA Informationis, 2023

## Requirements
- Python 3.6
- PyTorch >= 1.8.0
- numpy
- prefetch_generator
- sewar
- scipy


## Train on CAVE
`python main.py`


## Test on CAVE
`python test.py`
